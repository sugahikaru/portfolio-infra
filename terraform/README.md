# Terraform 構成
このディレクトリには、Terraform を使った AWS インフラ構築コードが含まれています。

- VPC、サブネット、EC2、RDS、ALB、WAF、、CloudWatchのリソース
- モジュール化された構成
- S3 バックエンドによる state 管理
